# toxic-comment-classifier

## Description

This project builds a model to detect toxic comments in text data.

## Data

* 159k comments
* Imbalanced dataset (toxic vs non-toxic ≈ 1:9)

## Approach

* Text cleaning
* Lemmatization using spaCy
* TF-IDF vectorization
* Models: Logistic Regression, Linear SVM, SGD

## Results

* Best model: Linear SVM
* F1-score: 0.80

## Tools

Python, scikit-learn, NLP
